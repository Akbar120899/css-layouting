# Display

  • setiap tag pada html berada di dalam sebuah kotak. seperti display pada css mengatur perilaku dari kotak tersebut
  • setiap tag pada html memiliki nilai default untuk property display. tapi kita juga dapat mengubah perilaku dari tag tersebut dengan mengganti valuenya

value dari display ini ada empat

  • inline
  • elemen yang tidak menambahkan baris baru ketika di buat
  • lebar dan tinggi elemennya sebesar konten yang ada di dalamnya
  • kita tidak dapat mengatur tinggi dan lebar elemen inline kecuali pada gambar
  • margin dan padding hanya mempengaruhi elelemen secara horisontal, tidak vrtikal
  • inline-block
  • tidak ada elemen yang secara default memiliki property display: inline-block
  • kita harus ubah secara manual property tersebut
  • perilaku dasarnya sama dengan elemen inline
  • perbedaannya, elemen inline-block ini dapat kita atur tinggi dan lebarnya
  • block
  • elelemen html yang secara default menambahkan baris baru ketika dibuat
  • jika tidak diatur lebarnya, maka lebar default elemen block akan memenuhi lebar dari browser/paretnya
  • kita dapat mengatur tinggi dan lebar dari elemen block
  • di dalam elemen block, kita dapat menyimpan tag dengan elemen inline, inline-block, atau bahkan elemen block lagi
  • none
  • diginakan untuk menghilangkan sebuah elemen

    • # Dimensi & Overflow

    A. dimensi

  • dimensi memiliki 2 properti
  • width : untuk menentukan lebar pada elemen
  • height : untuk menentukan tinggi pada elemen

    • B. Overflow

    overflow untuk mengatur elemen yang tidak cukup tempatnya dari parentnya

    overflow di gunakan ketika widt atau height tidak absolute/tidak relatif terhadap konten yang ad di dalamnya. lebih tepatnya jika konten tidak muat dari parentnya

    ada empat properti di dalam overflow

  • visible
  • ini adalah default overflow.
  • kalau ad elemen yang keluar dari parentnya maka akan di perlihatkan atau menyesuaikan
  • auto
  • css akan secara otomatis scrool ketika kontennya sudah tdk cukup lagi
  • jika kontenya cukup maka kontenya hilang secara otomatis
  • hidden
  • jika konten melebihi batas parentnya maka konten tersebut akan hilang/ tidak terlihat
  • scroll
  • scroll mirip seperti auto sama-sama memunculkan scroll ketika konten melebihi parentnya.tapi jika konten tidak melebihi paretnya maka scrollnya akan tetap ad

    • # Box Model

    • setiap elemen di halaman berad didalam sebuah box (kotak)
    • kita dapat mengatur ukuran dan posisi pada kotak tersebut
    • kita bisa memberi warna / gambar sebagai background kotak tersebut

    css box-model itu terdiri dari 4 bagian

  • margin
  • area transparan di sekitar kotak (diluar border)
  • border
  • batas di sekeliling konten dan padding
  • kita dapat mengatur warna dan ukuran
  • padding
  • area transparan di dalam kotak (antara konten dan border)
  • konten
  • konten sebenarnya di dalam box, bisa berupa teks atau gambar

    • catatan : yang dapat kita atur propertinya menggunakan css box-model hanya tiga yaitu Margin, Padding, dan border

    A. property untuk margin

    a. cara pakai margin

  • margin-top
  • untuk mengatur ukuran bagian atas
  • margin-right
  • untuk mengatur ukuran bagian kanan
  • margin-bottom
  • untuk mengatur ukuran bagian bawah
  • margin-left
  • untuk mengatur ukuran bagian kiri

    • b. overlapping margin

    • jika vertikal margin tidak saling menambahkan, hanya mengambil ukuran yang paling besar saja
    • margin dapat di berikan nilai negatif (-)
    • jika horizontal margin saling menambahkan
    • margin juga dapat di berikan value auto untuk membawa sebuah elemen ke tengah dari pembungkusnya

    B. property untuk padding

    a. cara pakai padding

  • padding-top
  • untuk mengatur ukuran bagian atas
  • padding-right
  • untuk mengatur ukuran bagian kanan
  • padding-bottom
  • untuk mengatur ukuran bagian bawah
  • padding-left
  • untuk mengatur ukuran bagian kiri

    • b. overlapping padding

    • padding tidak memiliki nilai negatif
    • padding tidak memiliki nilai auto
    • padding mempengaruhi ukuran box yang di buat

      • catatan : hati-hati menggunakan pedding karna mempengaruhi ukuran box

    C. property untuk border

    a. cara pakai border

  • border-top
  • untuk mengatur ukuran bagian atas
  • border-right
  • untuk mengatur ukuran bagian kanan
  • border-bottom
  • untuk mengatur ukuran bagian bawah
  • border-left
  • untuk mengatur ukuran bagian kiri
  • border memiliki tiga nilai
  • width : untuk ukuran
  • style : untuk jenis garis
  • color : untuk warna

    • b. untuk style ada beberapa yang bisa di pakai

  • style
  • solid / garis
  • dotted / titik-titik
  • dashed / garis putus-putus
  • double / garis berlapis dua

    • untuk mengatur ukuran kotak agar menyesuaikan terhadap ukuran padding atau border kita dapat menambahkan priperti

    • box-sizin: border-box
    • box-sizing di tambahkan ke elemen yang memiliki border atau padding

    # Float

    A. definisi float

    float merupakan properti pada css untuk mengatur posisi sebuah elemen. sebuah elemen dapat dipaksa untuk berada di sebelah kiri atau kanan dari parentnya/pembungkusnya dengan menggunakan proprty ini.

    B. value pada float ada tiga

  • float
  • none (ini value default dari float)
  • left (ini value untuk membawa elemen ke sebelah kiri parentnya)
  • right (ini value untuk membawa elemen ke sebelah kanan parentnya)

    • C. cara menggunakan

  • jika suatu elemen di berikan float maka elelemen yang berada di bawahnya akan menggantikan posisinya
  • jika ingin membuat elemen berjejeran maka harus di berikan float pada elemen yang berada dalam satu kotak

    • D. yang bisa dilakukan menggunakan properti foalt

  • text wrapping
  • untuk membuat text mengelilingi gambar/elemen lain
  • image gallery
  • membuat serangkaian gambar menjadi gallery yang rapih
  • multi-colums layout
  • membuat halaman menjadi beberapa kolom

    • # Position

  • css position ini dapat kita gunakan mengatur tata letak suatu elemen sesuai yang kita inginkan
  • css position hampir mirip dengan margin tetapi margin ketika digunakan elemen di bawahnya juga ikut bergerak jika di gerakkan ke atas/ke bawah
  • sedangkan pisition kita dapat menggerakkan elemen tampa mempengaruhi elemen di sekitarnya

    • A. value pada properti position

  • value dari properti position ada empat bagian
  • static
  • relative
  • absolute
  • fixed

    • B. keterangan value pada position

  • static
  • jika kita tidak memberikan properti position pada sebuah elemen maka default dari properti tersebut adalah static
  • value selain static
  • menggunakan position selain static (non-static). akan membuat sebuah elelemen menjadi seolah-olah berbeda dimensi dari eleme lainnya (elelemen melayang)
  • elelemen yang diberikan position selain static dapat menggunakan properti top, right, bottom, dan left untuk mengatur posisinya

    • a. relative

  • ketika kita menggerakkan elemen dengan posisi relative (menggunakan properti top, right, bottom, left), ruang yang ditempati oleh elemen tersebut akan masih ada
  • ketika kita menggerakkan elemen dengan posisi relative, elemen akan bergerak relatif terhadap posisi semulanya
  • jika kita memberikan properti top :0 ; dan left :0 ; maka elemen tidak berubah posisinya
  • sebaiknya relative digunakan saat kita hanya ingin memajukan sebuah elemen dan tidak menggerakkannya

    • b. absolute

  • ketika kita menggerakkan elemen dengan posisi absolute (menggunakan properti top, right, bottom, left), ruang yang ditempati oleh elemen tersebut dianggap tidak ada
  • ketika kita menggerakkan elemen dengan posisi absolute (menggunakan properti top, right, bottom, left), ruang yang ditempati oleh elemen tersebut dapat digantikan oleh elemen lainya
  • ketika kita menggerakkan elemen dengan posisi absolute, elemen akan bergerak relatif terhadap posisi parentnya/konteinernya
  • ketika kita menggerakkan elemen dengan posisi absolute,jika kita tidak ingin elemen tersebut keluar dari parentnya. parentnya juga kita harus berikan posision relatif (agar nyangkut)

    • c. fixed

  • kietika kita memberikan elemen dengan posisi fixed, maka elemen tersebut tidak akan berpindah tempat walaupun layar monitor di scroll

    • C. z-index

  • z-index digunakan untuk mengatur tumpukan pada elemen yang diberikan position

    • catatan : ketika kita memberikan posisi non-statik dan menggunakan properti (top, right, botton, dan left) properti tersebut dapat di berikan nilai negative (-)
